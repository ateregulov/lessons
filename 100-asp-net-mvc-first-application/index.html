Введение в веб-программирование

Фреймворк ASP.NET MVC

  • Запускаем студию, создаем новый проект с помощью пункта меню File -> New Project
  • На вкладке Visual C# -> Web выбираем проект ASP.NET MVC 4 Web Application.
  • В строке названия проекта Name вводим «Calc» и нажимаем «ОК».
  • Выбираем тип шаблона, Нам нужен Internet Application







  • Теперь, по нажатию клавиши F5 проект можно запустить, и мы должны увидеть в браузере картинку как на слайде
